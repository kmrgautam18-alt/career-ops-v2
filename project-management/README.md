# Project Management

This directory contains all project planning, sprint tracking, architectural decisions, release planning, and technical debt for Career-Ops v2.

## Purpose

This folder helps manage the project using an engineering-driven development process instead of ad-hoc coding.

## Contents

- Roadmap
- Sprint Backlog
- Sprint Reviews
- Architecture Decisions
- Technical Debt
- Release Planning
- Milestones

## Workflow

Every sprint follows this lifecycle:

1. Planning
2. Design
3. Development
4. Testing
5. Documentation
6. Git Review
7. Commit
8. Push
9. Sprint Closure
